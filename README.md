# Controls

Xbox controller:

* A: Select
* Y: Center map on current character
* B: Close/Cancel
* D-pad: move tile selector

Keyboard:

* Enter: Select
* Space: Center map on current character
* Escape: Close/Cancel
* Arrows: move tile selector

Alternative Keyboard:

* X: Select
* C: Center map on current character
* Z: Close/Cancel
* H/J/K/L: move tile cursor
